# Linux Commands — Week 4 Notes

1. `ls -la` — lists all files in a directory, including hidden ones, with permissions and ownership details.
2. `chmod` — changes the read/write/execute permissions on a file or directory.
3. `chown` — changes the owner and group of a file or directory.
4. `ps aux` — shows all running processes on the system along with resource usage.
5. `top` — displays a live, updating view of system processes and resource consumption.
6. `grep` — searches text for lines matching a pattern, useful for filtering logs and output.
7. `systemctl` — starts, stops, enables, and checks the status of system services.
8. `df -h` — shows disk space usage for mounted filesystems in human-readable format.
